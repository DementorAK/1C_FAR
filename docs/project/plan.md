# План инициализации проекта: 1C Far Plugin

## Обзор

Документ описывает план подготовки и реализации мультиплатформенного плагина для FAR Manager 3 / far2l / far2m, обеспечивающего прозрачную работу с бинарными артефактами платформы 1С:Предприятие 8.

**Язык реализации:** Rust с C-совместимым интерфейсом (C ABI) для Far Plugin API

**Целевые платформы:** Windows (FAR Manager 3, feature `far3`), Linux (far2l/far2m, feature `far2`)

**Основные источники:**

- [processing.md](../1C/processing.md) — внутренняя спецификация алгоритмов парсинга CF
- [formats.md](../1C/formats.md) — форматы .1cd, .cf, .cfe, .epf, .erf
- [bytecode.md](../1C/bytecode.md) — справочник по байт-коду
- [crossplatform.md](crossplatform.md) — анализ совместимости API FAR 2/3/far2l/far2m
