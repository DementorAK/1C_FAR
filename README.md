# 1C Far Plugin (`far1c`)

Мультиплатформенный плагин для FAR Manager / far2m для работы с артефактами 1С:Предприятие 8.

## Поддерживаемые форматы

| Формат | Описание | Чтение | Запись |
|--------|----------|--------|--------|
| `.epf` | Внешняя обработка | ✓ | ✓ |
| `.erf` | Внешний отчёт | ✓ | ✓ |
| `.cf` | Конфигурация | ✓ | ✓ |
| `.cfe` | Расширение конфигурации | ✓ | ✓ |
| `.1cd` | Файловая база данных | ✓ | ✓ |

## Установка

### Windows (FAR Manager 3)

1. Собрать плагин: `cargo build --release`
2. Скопировать `target/release/far1c.dll` в `%FARHOME%\Plugins\far1c\`
3. Перезапустить FAR Manager
4. Плагин появится в меню `F11`

### Linux (far2m)

1. Собрать плагин: `cargo build --release`
2. Скопировать `target/release/libfar1c.so` в `~/.config/far2m/Plugins/far1c/`
3. Перезапустить far2m

## Использование

### Просмотр артефакта

```
Ctrl+PgDn на .epf/.cf/.1cd  →  открывает виртуальную панель
Enter                            →  открыть содержимое
Backspace                        →  вернуться на уровень выше
```

### Редактирование

```
F3  →  просмотр (viewer)
F4  →  редактирование (editor)
F5  →  копировать из артефакта
F6  →  копировать в артефакт
F8  →  удалить
```

### Распаковка

```
Shift+F2  →  распаковка с выбором режима
```

### Сравнение

```
Ctrl+F3  →  сравнить с... (для .1cd)
```

## Режимы распаковки

| Режим | Описание |
|-------|----------|
| Raw | Бинарные блобы |
| Full-parse | Структурированные файлы |
| V8Unpack-style | Файлы `.header` + `.data` |
| Saby-style | Читаемая структура для Git |

## Требования

- FAR Manager 3 (Windows) или far2m (Linux)
- Rust toolchain (для сборки)

## Разработка

```bash
# Структура проекта
src/
├── lib.rs                       # Точка входа, экспорт Far Plugin API (C ABI)
├── far/                         # СЛОЙ 1: Взаимодействие с Far Manager
│   ├── mod.rs                   # Инициализация, настройки, диспетчеризация вызовов
│   ├── panels.rs                # Логика виртуальных файловых панелей (VFS для артефактов)
│   └── ui.rs                    # Отрисовка элементов UI (диалоги, прогресс-бары, меню)
├── v8_artifacts/                # СЛОЙ 2: Семантика и структура артефактов 1С
│   ├── mod.rs
│   ├── cf.rs                    # Высокоуровневая обработка контейнеров CF/EPF/ERF
│   ├── metadata.rs              # Навигация и разбор дерева метаданных конфигурации
│   └── onecd.rs                 # Логика парсинга внутреннего формата файловых баз .1cd
└── base/                        # СЛОЙ 3: Низкоуровневые библиотечные функции (I/O, парсинг)
    ├── mod.rs
    ├── reader.rs                # Абстрактный ридер (буферизованное чтение, работа с диском/памятью)
    ├── parser.rs                # Низкоуровневый парсер скобочных структур 1С
    └── utils.rs                 # Утилиты (нормализация путей, кодировки, временные файлы, deflate)

# Тестирование
tests/
├── epf/              # Тестовые .epf
├── erf/              # Тестовые .erf
├── cf/               # Тестовые .cf
├── cfe/              # Тестовые .cfe
└── 1cd/             # Тестовые .1cd
```
